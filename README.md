# Willans’ Formula Prime Computation

An experimental exploration of Willans’ closed‑form prime generator, including optimized evaluation, primality detectors, runtime benchmarking, and visual comparison.

## Repository Contents

- `willans.py` — Core implementation of Willans’ formula (prompts user to input **n**)
- `willans_detectors.py` — Prime detectors (trial division, Wilson’s theorem)
- `detector_comparison_script.py` — Prompts user for **max n** and compares detectors
- `detector_comparison.png` — Output graph generated by the comparison script
- `complexity analysis of willan's.pdf` — Analytical derivation of time complexity

---

## Prerequisites

- **Python 3.x**
- (Optional) **matplotlib** for graph generation

Install (optional):

```bash
pip install matplotlib
```

## Running the Code

### 1. Running Willans’ Formula (Interactive Input)

`willans.py` **asks the user to enter n**, the index of the prime to compute.

Run:

```bash
python willans.py
```

You will see:

```
Enter n:
Method ('wilson' or 'trial'):
```

Enter a value such as:

```
Enter n: 100
Method ('wilson' or 'trial'): wilson
```

This computes the 10th prime using Willans’ formula.

---

### 2. Running Detector Comparison (Interactive Input)

`detector_comparison_script.py` also **prompts the user** to enter the maximum n for benchmark comparison.

Run:

```bash
python detector_comparison_script.py
```

You will see:

```
Benchmark upto n:
```

Example:

```
Benchmark upto n: 700
```

The script will:

- Display a timing table for each detector  
- Generate `detector_comparison.png` (time vs n)
